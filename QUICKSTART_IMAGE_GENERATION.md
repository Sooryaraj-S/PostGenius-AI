## 🤖 Add AI Image Generation (Optional - 2 minutes)

### Get Free API Key from Replicate
1. Go to https://replicate.com/signin
2. Sign up (Free - takes 1 minute)
3. Copy your API key from https://replicate.com/account/api-tokens

### Activate in Dashboard
1. Change **Image Source** dropdown to "🤖 AI Generated (Replicate)"
2. Paste your API key
3. Click Generate
4. Wait 30-60 seconds for AI to create custom images ⏳

## 🐛 Troubleshooting

**Q: "Image generation failed" message?**
- Check internet connection
- Try "Free Stock Photos" instead
- Refresh the page

**Q: Replicate images take too long?**
- Expected! AI takes 30-60 seconds
- Meanwhile, system shows loading progress
- First image takes slightly longer

**Q: Lost my API key?**
- Go to https://replicate.com/account/api-tokens
- Generate a new one (takes 5 seconds)
- Paste it back into dashboard

**Q: Want different image?**
- Click "🔄 Regenerate" on output card
- For AI images: generates brand new with same caption
- For stock photos: fetches different from Unsplash
